# PropEditorX Improvements Backlog

These are the proposed ideas to enhance the Eclipse Plugin functionality.

| ID  | Feature                           | Description                                                                                                                                                                                                   | Status      |
| --- | --------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------- |
| 1   | Bidirectional Navigation          | **Hyperlinking (Ctrl + Click)** from Java to Properties and **Find Usages** from Properties to Java.                                                                                                         | **Planned** |
| 2   | Refactoring Support               | **Rename Participant**: Automatically update Java string references when a property key is renamed in the editor.                                                                                             | Backlog     |
| 3   | Dual-Mode Editing                 | Add a **Table/Grid View** tab alongside the Source View for easier bulk editing of keys and values.                                                                                                          | Backlog     |
| 4   | I18n Intelligence & Quick Fixes | **Bundle Validation**: Detect missing keys across translations. **Quick Fixes**: One-click solutions for duplicates or placeholders. **Placeholder Validation**: Verify `{0}, {1}` usage against Java calls. | Backlog     |
