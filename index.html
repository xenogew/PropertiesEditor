PropEditorX

Eclipse Update Site

This is the official P2 repository for PropEditorX, the modernized Java properties editor.

How to Install in Eclipse:

  1. Open Eclipse IDE.
  2. Go to Help > Install New Software...
  3. Copy and paste this URL into the 'Work with' field:
    https://xenogew.github.io/PropertiesEditor/
  4. Press Enter, select PropEditorX, and follow the prompts.
